
To run the program input the following commands in the terminal (UBUNTU) in /burgerproject/my_burger.

1. docker-compose build
2. docker-compose up


To run the test program input the following commands in the terminal (UBUNTU) in /burgerproject/my_burger.

1. docker-compose build
2. docker-compose run test

For more information about the tests you can check /docs/test files. You will find the test_plan, test_summary.
